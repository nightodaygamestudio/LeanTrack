LeanTrack

Minimalist fitness PWA for tracking weight, calories, water, and protein.
100% local, offline-first, no account required. Open source (MIT).

LeanTrack focuses on clarity and measurable progress.
No cloud. No subscription. No analytics scripts. Your data stays on your device.


Core Value Proposition

LeanTrack delivers:
- 100% local data (no account)
- Zero friction UX
- Transparent open source code (MIT)
- Works on any device (installable PWA)

Industry standard apps do:
- Forced cloud storage
- Ads, paywalls, sign-up walls
- Closed-source tracking
- Store lock-in (iOS/Android)

Measure. Adjust. Improve.
Progress requires visibility — LeanTrack makes the numbers obvious.


Features

- Installable as PWA (acts like native app)
- Offline-first and data stored locally on device
- Water tracking with monthly heatmap
- Daily weight tracking with trend line (7-day moving average)
- Calories and protein tracking including deficit
- ICS calendar reminders (no push server required)
- Light / Dark / System theme
- Export / Import data as JSON
